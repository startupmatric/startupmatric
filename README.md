## Projects

### [LLM Regression Engine](https://github.com/startupmatric/LLM-Regression-Engine)
CI-style evaluation system that detects hallucinations, behavioral drift, and prompt regressions across model versions before they hit production.

- Multi-model test suite runner (GPT-4o, Claude 3.5, Llama 3) with configurable evaluation strategies
- Automated regression scoring: semantic drift via cosine distance, LLM-as-judge scoring, assertion pass rate
- Real-time observability dashboard with per-run diffs and alert webhooks
- Catches regressions in under 90s across model comparisons

**Stack:** Python · FastAPI · PostgreSQL · Redis · Docker · Prometheus

---

### [BaseLock](https://github.com/startupmatric/BaseLock)
Automates PostgreSQL Row-Level Security (RLS) policy generation and validation using LLMs, with a self-healing pipeline that catches and rolls back bad policies before they reach production.

- LLM-generated RLS policies with automated validation and attack simulation
- Auto-fix and rollback pipeline for failed or regressed policies
- Policy versioning and audit trail for multi-user environments
- Simulates multi-role access patterns to surface privilege escalation risks

**Stack:** Python · PostgreSQL · LangChain · Docker · Pytest

---

### [FinPilot](https://github.com/startupmatric/finpilot-ai)
Agentic financial analysis backend using LangGraph for multi-step reasoning over financial data, with vector-based semantic search and role-based access control.

- LangGraph agent pipeline powered by Groq / Llama 3.1 for structured financial reasoning
- FastAPI backend with PostgreSQL, Redis caching, and vector search
- Role-based access control for multi-tenant data isolation

**Stack:** FastAPI · LangGraph · Groq / Llama 3.1 · PostgreSQL · Redis
